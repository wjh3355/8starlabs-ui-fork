## API Reference

| Prop              | Type                                             |
| ----------------- | ------------------------------------------------ |
| `system`          | `"SMRT"` (required)                              |
| `stationCode`     | `string \| string[]` (required)                  |
| `stationName`     | `string`                                         |
| `showStationName` | `boolean` (default: `false`)                     |
| `size`            | `"xs" \| "sm" \| "md" \| "lg"` (default: `"md"`) |
| `className`       | `string`                                         |
