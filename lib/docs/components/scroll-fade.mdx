```tsx showLineNumbers

```

```tsx showLineNumbers
<ScrollFade>
  {/** overflowed content here */}
  <div className="flex min-w-[700px]"></div>
</ScrollFade>
```

## API Reference

| Prop            | Type                                   | Default        | Description                                                               |
| --------------- | -------------------------------------- | -------------- | ------------------------------------------------------------------------- |
| `children`      | `React.ReactNode`                      | -              | **Required.** The content to be rendered inside the scrollable container. |
| `axis`          | `"horizontal" \| "vertical" \| "both"` | `"horizontal"` | Optional. Scroll direction(s) to enable and apply fade indicators         |
| `hideScrollbar` | `boolean`                              | `true`         | Optional. Hides the native scrollbar while keeping scroll functionality.  |
| `className`     | `string`                               | -              | Optional. Additional classes applied to the scroll container.             |
