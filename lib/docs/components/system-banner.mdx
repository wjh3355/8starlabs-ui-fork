## Usage

```tsx showLineNumbers
import SystemBanner from "@/components/ui/system-banner";
```

<Callout variant="info">
  Note: The `SystemBanner` component is designed to be used at the root level of
  your application, typically within your main `layout.tsx` or `_app.tsx` file,
  to ensure it appears above all other content.
</Callout>

```tsx {9-14} showLineNumbers title="app/layout.tsx"
export default function RootLayout({
  children
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <SystemBanner
          text="You are in Development Mode"
          color="bg-orange-500"
          size="sm"
          show={process.env.NODE_ENV === "development"}
        />
        {children}
      </body>
    </html>
  );
}
```

## API Reference

| Prop    | Type                           | Default              | Description                                   |
| ------- | ------------------------------ | -------------------- | --------------------------------------------- |
| `text`  | `string`                       | `"Development Mode"` | Optional. The text to display in the banner.  |
| `color` | `string`                       | `"bg-orange-500"`    | Optional. The background color of the banner. |
| `size`  | `"xs" \| "sm" \| "md" \| "lg"` | `"xs"`               | Optional. The size of the banner.             |
| `show`  | `boolean`                      | `true`               | Optional. Whether to show the banner.         |
