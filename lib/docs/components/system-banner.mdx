## Usage

```tsx showLineNumbers
import SystemBanner from "@/components/ui/system-banner";
```

<Callout variant="info">
  Note: The `SystemBanner` component is designed to be used at the root level of
  your application, typically within your main `layout.tsx` or `_app.tsx` file,
  to ensure it appears above all other content.
</Callout>

```tsx showLineNumbers
// In layout.tsx or _app.tsx
export default function RootLayout({
  children
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <SystemBanner
          text="You are in Development Mode"
          color="bg-orange-500"
          size="sm"
          show={process.env.NODE_ENV === "development"}
        />
        {children}
      </body>
    </html>
  );
}
```

## API Reference

| Prop    | Type                                             |
| ------- | ------------------------------------------------ |
| `text`  | `string` (default: `"Development Mode"`)         |
| `color` | `string` (default: `"bg-orange-500"`)            |
| `size`  | `"xs" \| "sm" \| "md" \| "lg"` (default: `"xs"`) |
| `show`  | `boolean` (default: `true`)                      |
